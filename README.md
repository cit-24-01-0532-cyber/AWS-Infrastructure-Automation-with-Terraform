# ☁️ AWS Infrastructure Automation with Terraform

This project demonstrates the creation of a secure, scalable AWS infrastructure using **Infrastructure as Code (IaC)** with Terraform.

## 🏗️ Architecture
The infrastructure includes:
* **Custom VPC** with a Public Subnet.
* **Internet Gateway** for external connectivity.
* **Security Groups** allowing SSH (22) and HTTP (80) traffic.
* **EC2 Instance** running Ubuntu with Nginx web server.

## 🛠️ Tech Stack
* **Cloud:** AWS
* **IaC Tool:** Terraform
* **Web Server:** Nginx
* **OS:** Ubuntu
